# Challenge-Alura-One
O decodificador de texto tem como finalidade criptografar e descriptografar mensagens a partir da modificação das vogais contidas nas palavras do texto.
Regra para criptografar:
Para letra 'a' = 'ai'
Para letra 'e' = 'enter'
Para letra 'i' = 'imes'
Para letra 'o' = 'ober'
Para letra 'u' = 'ufat'
Regra descriptografar:
Para letra 'ai' = 'a'
Para letra 'enter' = 'e'
Para letra 'imes' = 'i'
Para letra 'ober' = 'o'
Para letra 'ufat' = 'u'
